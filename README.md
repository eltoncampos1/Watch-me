### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/eltoncampos1/Watch-me.git
   ```

2. enter the project folder
   ```sh
   cd file_name
   ```

3. Run server
   ```sh
   npm or yarn dev
   ```
 4. Run the Json-server
  ```sh
     yarn server
  ```
    
  ```sh
     npm server
  ```
  
 5. Run the aplication
  ```sh
     yarn dev
  ```
    
  ```sh
     npm dev
  ```
